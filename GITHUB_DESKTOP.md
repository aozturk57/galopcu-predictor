# GitHub Desktop ile Yükleme (TOKEN GEREKMEZ!)

## GitHub Desktop Kurulumu

1. **GitHub Desktop İndir:**
   - https://desktop.github.com adresine git
   - "Download for macOS" butonuna tıkla
   - İndir ve kur

2. **GitHub Desktop'u Aç:**
   - GitHub hesabınla giriş yap (`aozturk57`)
   - Otomatik olarak token oluşturur, token'a gerek yok!

3. **Projeyi Aç:**
   - GitHub Desktop'da "File" → "Add Local Repository"
   - Şu klasörü seç: `/Users/ardaozturk/galopcu_predictor web calisan 7 model kopyası 2`
   - "Add repository" butonuna tıkla

4. **GitHub'da Repo Oluştur:**
   - GitHub Desktop'da "Repository" → "Publish repository"
   - Name: `galopcu-predictor`
   - Description: (boş bırak)
   - "Keep this code private" işaretini KALDIR (Public olsun)
   - "Publish repository" butonuna tıkla

5. **BİTTİ!** ✅
   - Dosyalar otomatik yüklenecek
   - Token'a gerek yok!

## VEYA: Web'den Manuel Yükleme

Eğer GitHub Desktop da istemiyorsan:

1. GitHub'da repo oluştur: https://github.com/new
   - Name: `galopcu-predictor`
   - Public seç
   - "Create repository"

2. GitHub'da "uploading an existing file" linkine tıkla

3. Dosyaları sürükle-bırak ile yükle
   - Tüm proje klasörünü seç
   - "Commit changes" butonuna tıkla

Bu yöntem daha uzun sürer ama token gerektirmez.
